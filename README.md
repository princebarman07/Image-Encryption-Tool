# Image-Encryption-Tool
A Python tool for secure image encryption using AES-128
## Features
* **AES-128 Encryption:** Uses the Advanced Encryption Standard for high-level security.
* **Pixel Shuffling:** Adds an extra layer of security by rearranging image pixels.
* **Key-Based Access:** Only users with the correct secret key can decrypt the image.
* **Lossless Quality:** The image quality remains intact after the decryption process.

## How to Use
1. **Clone the Repo:** `git clone https://github.com/princebarman07/Image-Encryption-Tool.git`
2. **Install Dependencies:** `pip install -r requirements.txt`
3. **Run the Script:** Execute the main Python file and follow the on-screen prompts to encrypt or decrypt your images.
